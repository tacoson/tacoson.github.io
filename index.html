換行字元

Unix系統裡，每行結尾只有“\n”；
Windows系統裡面，每行結尾是即“\n\r”；
Mac系統裡，每行結尾是“\r”。
一個直接後果是，Unix/Mac系統下的文件在Windows裡打開的話，所有文字會變成一行；而Windows裡的文件在Unix/Mac下打開的話，在每行的結尾可能會多出一個^M符號。

10分鐘週記 - SFTP研究番外篇

2022-12-14

SFTP番外篇

近期主要因為需要研究SFTP SERVER而起，SFTP(SSH File Transfer Protocol AKA. Secure FTP)，主要是透過SSH登入作業時，會進行加密，使得竊聽者無法竊取密碼
主要對話通道會進行作業

  • 建立和終止階段作業
  • 識別、導覽和列出目錄的內容
  • 將檔案移入/移出目錄及在目錄內移動檔案
  • 刪除檔案

實務上，SSH(port22)也常用來取代telnet，他可以透過產生SSH host key pair的方式，透過提供SERVER public key 取代密碼進行登入，且無有效期限設計，不用每次都需要輸入密碼
產生的金耀，用工具讀取需要輸入的密碼稱之 passphrase

番外篇，大部分都在研究 CRUSHFTP 額外產生網路搜尋的

數位簽章與憑證

數位簽章(DS, Digital Signature)
傳送端本質上是透過hash[訊息]的方式計算出hash值，將hash加密後變成[數位簽章]和[訊息]傳送給接收端;接收端則將[訊息]和[數位簽章]分離，針對[數位簽章]進行解密，得hash值，和[訊息]計算出來的hash值進行比對得知其完整性及不可否認性
實務上，傳送與接收，會將資料和數位簽章再次進行加密

數位憑證(Digital Certificate)
概念上是一種由公信力機構發行的證明文件，說白一點就是記載姓名、地址、電子郵件、公鑰，並且具有有效期限，概念上與身分證件一樣，對於擁有者來說，CA(憑證授權)中心會負責賦予公鑰與私鑰，並提供公鑰公信力，取得信任。
參考網址: 數位簽章與憑證

GIT

參考網址: GIT ebook

GIT 已經紅滿久的，10年前對他的概念只知道是分散式的版控軟體，但對於實際上的分支系統設計並不清楚，官網的說明文件寫得滿有知識含量，值得一看
目前我只有看第2章Git基礎及第3章使用GIT分支，先把看到的指令做個剪貼本:

  • 檔案放置在有Working Directory(WD), Staging Area(預存區), .git repository(Repository)，三個區域

    檔案狀態
    Modified 檔案已經變更但是尚未存於本地端資料庫
    Staged 已預存這檔案會被存在下次快照
    Commited 已儲存至本地端(local)資料庫

  • 預存區實體是單一檔案

  • 基本 Git流程，在工作目錄修改檔案；預存檔案並將快照新增至預存區；做提交動作，這會讓預存區的檔案快照永久存在Git目錄中

  • Git的分支模型和其他 (Version Control System, 以下簡稱VCS) 與眾不同的地方，便是每次新增的快照都會以”增加”存在，而並非以DIFF方式，大幅度降低本地資料庫故障機率。

  • 單次提交的目錄結構(Tree)，也可視為快照，有使用SHA當雜湊碼物件識別

  • 提交和親代提交的關係(注意master和HEAD指標意義)

  • Master是第一個建立的分支；HEAD並非像其他CVS下git branch指令會切換”目前”分支。

早上工作中發現不對勁，緊急返家並到診所掛號拿藥，由於公司仍有案子正在忙，稍作休息下午還是到公司上班。除了頭脹外，印象很深刻是女聲在我的右耳會有變聲的感覺(應該是回音共鳴)，在辦公室環境還算是可以忍受，我最不能忍受是走在馬路上車水馬龍的摩托車引擎聲。

由於到晚上還沒好，而且我的症狀比較像耳咽管開放症，確切原因不清楚，因此當晚到診所再詢問一次意見並請醫生開轉診單，想做進一步檢查。
雖然很慌張，後續是打算先把醫師開的三天藥吃完後再說，應該還是會去教學醫院檢查一次。

症狀出現後第三日

症狀每天睡醒之後都有逐步減輕，自我觀察我身體上除了耳朵不舒服的狀況外，伴隨右邊肩膀一路延伸到右後腦杓都產生緊繃劇痛感，臉部肌肉也很沒有力氣。
這期間我做的就是睡足，吃的部分就是每天奇異果或者富士蘋果，正餐保持少油且增加好的蛋白質來源，比較特別的是由於中醫提到耳鳴都會順便講到腎虛，因此每天來蚵仔或蛤蜊湯，另外強化水分攝取，保持鼻腔的濕潤。
每天河堤快走流汗一下，腹式呼吸法讓自己身體的氣可以順暢點(畢竟平常都戴著口罩)，運動同時要同時擺手，有點像是太極，口訣是:想像一下自己很像大海，雙臂是海浪。

雖然還沒完全好，但我覺得和過度疲勞脫離不了關係，過度疲勞也是也是自律神經失調，氣血不順身體並不聽從指揮。
希望這次不要有後遺症!!

10分鐘週記-Krita筆記

2022-07-24

給自己一個挑戰，10分鐘來完成紀錄一週大小事，簡短紀錄一下筆記:

教學影片

參考網址: HOW TO PAINT IN KRITA
參考網址: Clipping Masks: Alpha inheritance, Alpha Lock & Groups in Krita

第一部影片Jenna Drawing 製作的影片，超清楚解釋如何使用Krita，主要是Step by step把從無到有的繪圖教學，開畫布、基本介面介紹，移動畫布、選擇筆刷、顏色選擇…等，後半段會實際演練出一部作品。
第二部影片是查資料找到的，專門介紹如果介紹什麼是Alpha(透明層)、怎麼進行Alpha鎖定並與群組圖層(Group)應用、Alpha繼承用途，最後有個人像繪製。

因對電繪懵懵懂懂，自己理解的部分
Alpha : 其實就是我們看到的非繪製的區域，說白一點是去背的區域
Alpha lock : Alpha區域不給你畫，這樣上色區域就可被限制，這是著色的一個好特性
Alpha繼承: 看起來可用於物體光影著色

實際操作

玩了一會兒，列舉一下有使用過的功能

  • 匯入影像: [圖層] → [匯入/匯出] → [匯入] → [作為顏料塗層]
  • 圖層: Alpha鎖定、GROUP Layer、Alpha繼承
  • 匯出 GIF: [檔案] → [繪製動畫]
  • 工具箱: 色彩取樣(P)、手繪筆刷工具(B)、移動工具(T)、填充工具(F)、選擇區域變形(CTRL+T)、剪裁工具(C)…等
  • 檢視畫布: [1]:100%原比例大小檢視 [2]:fit顯示區域的檢視 [3]鼠標所在位置的檢視 [4]左邊選轉 [5]擺正 [6]右邊旋轉 [m]左右邊鏡像檢視
  • 筆刷: 整理喜歡的筆刷到 My favorite
